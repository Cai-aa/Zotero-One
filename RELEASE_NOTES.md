# 🎉 Zotero One v0.1.0 - Initial Release

We're excited to announce the first release of **Zotero One**, an enhanced productivity plugin that brings item numbering and quick preview capabilities to your Zotero workflow!

## 🌟 What's New

### 📋 Item Numbering
Transform your Zotero library with sequential numbering that makes referencing and organizing your research easier than ever:

- **Smart Numbering**: Automatic sequential numbers for every item
- **Collection-Aware**: Independent numbering for different collections
- **Dynamic Updates**: Numbers automatically adjust when you add, remove, or reorganize items
- **Persistent Settings**: Your column position and sorting preferences are remembered
- **Easy Toggle**: Quick on/off switching via right-click or toolbar

### 👁️ Quick Preview
Get instant PDF previews without the wait:

- **Lightning Fast**: Preview PDFs directly in Zotero
- **Performance Optimized**: Smart loading of first 10 pages
- **Keyboard Friendly**: Simple ESC-to-close functionality
- **Robust Fallback**: Graceful handling when PDFs aren't available

### 🎨 Modern Interface
- **Seamless Integration**: Fits naturally into Zotero's interface
- **Beautiful Icons**: Custom-designed icons reflecting the plugin's functionality
- **Bilingual Support**: Full English and Chinese localization
- **Customizable**: Adjust column positions to match your workflow

## 🚀 Quick Start

1. **Download**: Get `zotero-one.xpi` from the [Releases page](https://github.com/user/zotero-one/releases)
2. **Install**: In Zotero, go to Tools → Add-ons → Install Add-on From File
3. **Restart**: Restart Zotero to activate the plugin
4. **Enjoy**: See the new numbering column and start using quick preview!
